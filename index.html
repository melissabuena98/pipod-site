A Collaborative Project Designed and Developed by

Cooper Astle, Melissa Buena, Nathaniel Delgado, Arica Rohwer, and Adrian Alvarado

What is Pi Pod?

The PiPod is a Raspberry Pi 3 that is built to resemble a modern iPod Touch. The device utilizes the Pi to run the various built-in applications and utilities that we will create. It also features an on board touch screen in which users will use to interact with apps on the PiPod. Its purpose is to provide basic smart device features to anyone without breaking the bank. It can serve as a back-up phone or tablet because of its easy capability to expand the piâ€™s storage amount.

FEATURES

User-based registration via name and email, as well as personal passcode to unlock.

Clock which displays local time.

Music app with GUI interface.

Weather app displaying 7 day local forecast.

Settings app allowing user to adjust registration settings as well as passcode configuration and background color.


Stretch Goals not met:

Video player

Camera function

PROJECT SUCCESSES

Division of tasks between members allowed features to be developed simultaneously.

PROJECT CHALLENGES

As a group, we all were not familiar with Python or TKinter, leaving us to dedicate time to learning.

AGILE PROCESS

Allowed team members to report daily progress on tasks during stand-ups.

Streamlined project workflow by dividing tasks into backlogs.
