PI-POD

A Collaborative Project Designed and Developed by

Cooper Astle, Melissa Buena, Nathaniel Delgado, Arica Rohwer, and Adrian Alvarado

What is Pi Pod?

The PiPod is a Raspberry Pi 3 that is built to resemble a modern iPod Touch. The device utilizes the Pi to run the various built-in applications and utilities that we will create. It also features an on board touch screen in which users will use to interact with apps on the PiPod. Its purpose is to provide basic smart device features to anyone without breaking the bank. It can serve as a back-up phone or tablet because of its easy capability to expand the pi’s storage amount.

FEATURES

User-based registration via name and email, as well as personal passcode to unlock.


Clock which displays local time.


Music app with GUI interface.


Weather app displaying 7 day local forecast.


Settings app allowing user to adjust registration settings as well as passcode configuration and background color.


TECHNOLOGIES

Trello, GitHub, and Discord for communication and collaboration.


MAPLE Stack (MySQL, AWS, Python, Linux, Express)


Raspberry Pi 3, with additional hardware.


Visual Studio Code as our IDE.


CHALLENGES

Persistent language barrier, as we all were not familiar with Python going into this project.


Hardware delays late into the project life caused minor setbacks.


Merging errors, which set us back on multiple occasions.


TKinter is awful.
